---

## ¿Qué es?

Tu Casa de Detalles es una tienda online argentina orientada a un público juvenil y femenino. Ofrece accesorios para el pelo, accesorios de invierno, morrales, artículos de librería, llaveros y productos de cuidado personal.

---

## Stack técnico

- **Frontend** — HTML, CSS y JavaScript vanilla, hosteado en GitHub Pages
- **Backend** — Google Apps Script como API REST
- **Base de datos** — Google Sheets
- **Imágenes** — Google Drive (CDN via lh3.googleusercontent.com)
- **IA** — Gemini 2.5 Flash para generación y análisis de descripciones de productos

---

## Funcionalidades

### Tienda pública
- Catálogo de productos con búsqueda y filtro por categoría
- Modo claro / modo oscuro con transición suave
- Carrito de compras con checkout por WhatsApp y MercadoPago
- Sección de novedades
- Diseño responsive (mobile y desktop)

### Panel de administración
- Gestión completa de productos con hasta 4 fotos por ítem
- Registro de ventas con selector de productos del catálogo o inventarios
- Registro de gastos con comprobantes
- Generación de recibos PDF por venta
- Resumen mensual con PDF descargable
- Cierre de mes: genera el PDF, guarda historial y limpia ventas/gastos
- Inventarios dinámicos con alertas de stock mínimo
- Integración con Gemini Vision para análisis automático de fotos de productos
- Calculadora de precios y márgenes
- Configuración de datos del negocio (WhatsApp, Instagram, MercadoPago)
